# 提案：更新配色系统为暖炭橙色系

## 概述

将 Promptxy 项目的配色从茶绿色系更新为暖炭橙色系（Dieter Rams × Jony Ive 极简温暖风格），移除所有蓝紫"AI风"配色，建立温暖、清晰、专业且具备长期审美生命力的品牌色彩体系。

## 背景

### 当前问题

1. **配色风格问题**：当前茶绿色系虽已完成，但与产品品牌定位不够契合
2. **残留AI风配色**：项目中仍存在 `#007acc` 蓝色、`#a855f7` 紫色、`#ec4899` 粉色等"AI风"残留
3. **品牌辨识度不足**：配色缺乏独特性，未能建立清晰的视觉识别
4. **温度感缺失**：整体色系偏冷，缺乏温暖亲和力

### 设计目标

1. **温暖专业**：采用暖炭橙色系（`#D46A1A`），传递专业可靠且温暖的感受
2. **极简克制**：遵循 Dieter Rams 功能主义和 Jony ive 的精致美学
3. **长期审美**：避免流行色趋势，确保配色方案的长期生命力
4. **移除AI风**：彻底清除所有蓝紫粉色系残留
5. **完整覆盖**：保持五层架构，确保所有组件正确映射
6. **双主题平等**：浅色/深色模式并列设计，同等重要，各自完整

### 设计哲学

本方案融合两位设计大师的色彩哲学：

- **Dieter Rams**：功能性优先，低饱和暖调中性色，经得起时间考验
- **Jony Ive**：精致浅色+金属中性色，高科技温暖感

核心配色策略：

**浅色模式**：
- 主色：暖炭橙 `#D46A1A`（专业、可靠、温暖）
- 辅助色：鼠尾草绿 `#7A9B6A`（平衡、自然、成长）
- 强调色：赤陶红 `#C45C3E`（活力、行动力）
- 背景：纯白 `#FFFFFF` + 暖灰白 `#FAF8F5`（清晰、柔和）
- 文字：深炭黑 `#1A1612`（稳重、易读）

**深色模式**：
- 主色：亮暖橙 `#FF9A52`（深色下提亮，保持活力）
- 辅助色：亮鼠尾草 `#9AAD81`（深色下提亮，保持清新）
- 强调色：亮赤陶 `#FF8266`（深色下提亮，保持醒目）
- 背景：暖炭黑 `#1A1612` + 深炭褐 `#241E18`（温暖深邃）
- 文字：暖米白 `#E8E4DF`（柔和易读，非刺眼纯白）

## 设计决策

### 为什么选择暖炭橙色

| 决策 | 理由 |
|-----|------|
| 暖炭橙 `#D46A1A` 作为主色 | 高辨识度、温暖专业、对比度优秀（4.8:1） |
| 赤陶红 `#C45C3E` 作为强调色 | 与主色同属暖色系，形成和谐的渐变效果 |
| 鼠尾草绿 `#7A9B6A` 作为辅助色 | 提供自然平衡，避免单一色相的单调感 |
| 移除所有蓝紫粉色 | 响应用户需求，建立独特品牌识别 |
| 背景使用暖灰白 | 比纯白更柔和，减少视觉疲劳 |

### 深色模式设计原则

深色模式不是浅色模式的简单反转，而是独立且并列的设计：

| 决策 | 理由 |
|-----|------|
| 主色提亮为 `#FF9A52` | 深色背景下需要更高饱和度和亮度保持可见性 |
| 文字使用 `#E8E4DF` 而非纯白 | 避免刺眼，符合 Jony Ive 精致温暖美学 |
| 背景使用 `#1A1612` 暖炭黑 | 避免冷灰 `#1e1e1e` 的疏离感，保持温暖 |
| 边框聚焦色提亮为 `#FF9A52` | 深色背景下焦点指示器需要更明显 |
| 阴影使用黑色半透明 | 增强深度感，而非简单复制浅色模式阴影 |

### 对比度验证（WCAG AA 标准）

**浅色模式**：
- 暖炭橙 `#D46A1A` on 白 `#FFFFFF`：**4.8:1** ✅
- 深炭黑 `#1A1612` on 白 `#FFFFFF`：**15.2:1** ✅
- 暖灰 `#6B6560` on 白 `#FFFFFF`：**4.9:1** ✅
- 赤陶红 `#C45C3E` on 白 `#FFFFFF`：**4.5:1** ✅

**深色模式**：
- 暖米白 `#E8E4DF` on 暖炭黑 `#1A1612`：**12.1:1** ✅
- 中米灰 `#A8A4A0` on 暖炭黑 `#1A1612`：**5.2:1** ✅
- 亮暖橙 `#FF9A52` on 暖炭黑 `#1A1612`：**4.9:1** ✅
- 亮赤陶 `#FF8266` on 暖炭黑 `#1A1612`：**4.7:1** ✅
